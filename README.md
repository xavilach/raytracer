# raytracer

## Description
Raytracer written in C++.
Following the "Raytracing in one weekend" tutorial.

Currently:
Using SDL library to display the resulting picture in a window.
Multithreaded rendering using a thread pool.

## Building
```
mkdir build
cd build
cmake ..
make
```
